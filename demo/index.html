    Preset
    Eleven themed presets across five behaviors — Snow and Rain ride the new FALL behavior. Nothing fell before v1.2. Live-tweak with the sliders below; switch behavior via a preset. --
    Parallax depth
    Every behavior already scales size, alpha and movement by a particle's z. Quantizing z at spawn is the parallax — no extra draw pass, no per-frame cost. Re-seeds the pool.
    Pointer
    One pass over the pool before behavior.tick — so every behavior gets this free, including custom ones. Force is depth-scaled: near particles shove hard, far ones barely move. Off under reduced motion.

    Theme registry
    THEMES is a registry, not a constant. Register a preset and it lands in THEME_META — the picker above is rebuilt straight from it, with no code change. That is the whole point.
    Worker mode --
    /worker is a separate export path: the core stays zero-dependency, and @zakkster/lite-worker is only fetched when you press this. The atmosphere below runs entirely off the main thread on an OffscreenCanvas.
